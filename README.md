# Musilinda – Theory You Can Feel

This is the static landing page for Musilinda, a music theory app that teaches by doing. Learn to sing, see, and feel music harmony through interactive lessons.

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```
2. **Start the development server:**
   ```bash
   npm run dev
   ```
3. **Build for production:**
   ```bash
   npm run build
   ```
   The static site will be output to the `dist` folder.

## Deployment

Host the contents of the `dist` folder on any static hosting service (e.g. Vercel, Netlify, GitHub Pages).
